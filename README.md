# CaliforniaHousePricePrediction

過程 (Process):
資料預處理: 載入加州房價資料集，對特徵和目標變數進行 Min-Max 標準化，並將資料分割為訓練集和測試集。
WAN MODULE 訓練與調整:
Weight_tuning_LG_UA: 訓練一個初始神經網路，使用自定義的學習率調整機制（UA Module），目標是使訓練集上的最大絕對誤差 (max_loss) 低於預設的門檻 (gate)。
Analogizing_Ri_RE_SO_MU: 如果初始模型未達標（max_loss 超過 gate），則嘗試透過新增隱藏節點並調整權重來修正模型，以處理造成高誤差的資料點。
Network_Tuning_4: 迭代地刪除模型中的隱藏節點（從對輸出權重影響最小的節點開始），並重新微調其餘權重，以在保持性能的同時簡化模型。
基準模型比較: 訓練並評估線性迴歸、SVM、決策樹和隨機森林模型。
模型評估: 使用平均絕對誤差 (MAE) 和誤差在 1.5 倍標準差內的機率來評估所有模型的性能。

方法 (Method): 程式碼採用了 TensorFlow/Keras 構建神經網路，並實現了獨特的自適應學習率 (UA Module)、增強網路容量 (Analogizing_Ri_RE_SO_MU) 和精簡網路結構 (Network_Tuning_4) 的客製化訓練策略。同時，也包含了多種機器學習演算法作為基準測試。

目的 (Purpose): 該程式碼旨在利用自定義的神經網路模型（稱為 WAN MODULE）來預測加州房價（median_house_value），並與其他基準模型（如線性迴歸、SVM、決策樹、隨機森林）進行比較。特別是，它探索了一種迭代式的模型訓練和結構優化方法。

結果 (Results):
自定義的 WAN MODULE 模型在測試集上的 MAE 為 7232.26。然而，其誤差在 1.5 倍標準差內的機率為 0.0，這表示模型在大多數情況下未能將誤差控制在期望範圍內。
基準模型在測試集上的 MAE 如下：
線性迴歸：50875.08
SVM：32059.28
決策樹：890.50
隨機森林：710.05
根據 MAE 來看，WAN MODULE 模型優於線性迴歸和 SVM，但在這個特定的資料集上，決策樹和隨機森林模型的性能顯著優於 WAN MODULE 模型，展現出更低的預測誤差。
